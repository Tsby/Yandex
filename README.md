# Yandex
## Пример кода
Файлы в данном репрезитории - мой код с практики по кластеризации и классификации рукописных цифр(выборка MNIST) на Python.
В ходе прохождения практики я изучил, применил и сравнил основные методы кластеризации и классификации выборки.
Код представлен в файлах с расширением .ipynb для просмотра в Jupiter Notebook.
